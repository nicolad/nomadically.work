# Agent Skills

This directory contains reusable skills that agents can use to perform specific tasks.

## What are Skills?

Skills are structured instructions that help agents perform common tasks consistently. They can include:

- Step-by-step procedures
- Best practices
- Common patterns
- Error handling guidelines

## Creating a Skill

Create a new file in this directory with a `.md` or `.txt` extension containing the skill instructions.

Example: `job-analysis.md`
